<h1 align="Center">Project in CCPROG3</h1>
<p align="Center">📚LifeClass Planner is a simple and user-friendly student planner designed to help students stay organized and never miss a deadline. It provides an all-in-one platform for managing academic responsibilities efficiently.</p>
<p align="Center">The system includes a Task Manager, Calendar & Deadlines, Progress Tracker, and an interactive Dashboard where users can view their tasks, schedules, and productivity at a glance.</p>
<p align="Center">Built using HTML, CSS, and JavaScript, this project was developed as an academic project by PlanForge to promote better time management and organization for students.</p>
